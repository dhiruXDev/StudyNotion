# StudyNotion

## Overview
StudyNotion is a fully functional Ed-Tech platform built using the MERN stack (MongoDB, ExpressJS, ReactJS, NodeJS). It enables users to create, consume, and rate educational content while providing a seamless and interactive learning experience for students and a platform for instructors to connect with learners worldwide.

## Features

### For Students
- Homepage with course listings and user details.
- Course details and wishlist management.
- Checkout and payment processing with Razorpay integration.
- Access to video lectures and additional resources.

### For Instructors
- Dashboard for course management and insights.
- Tools to create, update, and delete courses.
- Media content management via Cloudinary.

### For Admins (Future Scope)
- Platform insights.
- Instructor and course management tools.

### Core Features
- User Authentication: Email/password, OTP verification, and password reset.
- Responsive Design: Optimized for various devices using TailwindCSS.
- REST API: For seamless communication between front-end and back-end.
- Secure Payments: Razorpay integration to manage payment lifecycle.

## Tech Stack
- **Front-end**: ReactJS, TailwindCSS, Redux, React Router.
- **Back-end**: Node.js, Express.js.
- **Database**: MongoDB.
- **Media Storage**: Cloudinary.
- **Other Tools**: Bcrypt, JWT, Razorpay.

## Installation

### Prerequisites
- Node.js
- MongoDB
- npm or yarn

 
 
## Deployment

### Hosting
- **Front-end**: Vercel.
- **Back-end**: Render or Railway.
- **Database**: MongoDB Atlas.
- **Media**: Cloudinary.

### Steps
1. Deploy the front-end via Vercel by linking the GitHub repository.
2. Deploy the back-end via Render or Railway with proper environment variables.
3. Configure MongoDB Atlas and Cloudinary.
